# GoldFolio

A premium Wealth Management Suite built with React, Vite, and Tailwind CSS.
Designed with a "Premium Gold" aesthetic to track your gold and stock portfolio with elegance.

## Features

- **Dashboard**: "Total Invested" stats, Top Assets, and a Portfolio Allocation Donut Chart.
- **Transactions**: Record and view history of Buy/Sell orders.
- **Profit Calculator**: Estimate your ROI and potential profit before you sell.
- **Gold Theme**: Custom design system using `#D4AF37` (Gold) and `#2C3E50` (Dark Slate).

## How to Run

The application is already set up. You can run it using:

1.  **Install Dependencies** (First time only)
    ```bash
    npm install
    ```

2.  **Start the Server**
    ```bash
    npm run dev
    ```

3.  **Open in Browser**
    Visit [http://localhost:5173](http://localhost:5173)

## Tech Stack

- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Charts**: Recharts
